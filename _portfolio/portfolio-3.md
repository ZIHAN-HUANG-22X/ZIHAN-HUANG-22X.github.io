# 项目介绍
本课程作业以2016-2023年CCTV“一带一路”新闻文本为研究对象，运用词共现网络、LDA主题模型、词向量聚类、面板数据回归多种方法，分析新闻话语的主题框架、国家叙事类型、时间演化规律，并实证检验新闻话语与双边经贸的关联及距离调节效应。
